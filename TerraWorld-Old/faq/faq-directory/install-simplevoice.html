Установка SimpleVoice

Обновленно : 1 Мая 2024 г. 
    Как установить SimpleVoice игроку:
    
    (Если у вас Original MineLauncher. В Пиратке уже Forge/fabric есть)
    
    Скачай и установи Minecraft Forge / Fabric в свой лаунчер
    
    Скачай мод  Forge 1.12.2
    
    Forge 1.16.5 / Fabric 1.16.5 = Fabric API 1.16.5
    Forge 1.20.1 / Fabric 1.20.1 = Fabric API 1.20.1
    Для Fabric надо Fabric API
    
    Не распаковывая, скопируй в .minecraft\mods
    Готово
    Чтобы открыть меню мода - нужно нажать кнопку V.
